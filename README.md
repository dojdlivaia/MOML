Код к визуализации для учебника "Математические основы машинного обучения".

## Структура проекта

Каждый класс рендерится в одно видео. Видео по названию соответствует классу. Видео сохраняются в папку с названием по названию файла. 

Видео сохраняются в папку [videos](https://github.com/koroteevmv/MOML/tree/main/media/videos)

Изображения (последний кадр видео) - в папку [images](https://github.com/koroteevmv/MOML/tree/main/media/images)

### Главные файлы

[vectors.py](https://github.com/koroteevmv/MOML/blob/main/vectors.py) - Глава "Вектора". Ранее назывался main, поэтому папки создавались с таким названием.

[matricies.py](https://github.com/koroteevmv/MOML/blob/main/matricies.py) - Глава "Матрицы"
